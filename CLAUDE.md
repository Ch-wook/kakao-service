# 🔒 보안 및 개인정보 보호 규칙

- 코드 내에 이메일, 비밀번호, API 키, 개인 식별 정보(PII) 등 개인정보가 포함되어 있다면 절대로 외부 로그로 출력하거나 저장하지 마세요.
- 더미 데이터가 필요할 때는 실제 개인정보 대신 무작위 가상 데이터(Faker)를 사용하세요.
- 이 계정은 여러 사용자가 공동으로 사용하는 계정입니다. 이전 대화나 작업 과정에서 입력된 특정 사용자의 개인정보(이름, 연락처, 개인 코드 등)가 있더라도, 이를 다른 사용자와의 대화나 답변에서 절대 발설하거나 인용하지 마세요.
- 작업 도중 발견되거나 임시 저장된 기존 개인정보가 있다면 즉시 기억에서 지우고(삭제), 파일이나 로그 등 그 어디에도 별도로 기록하거나 저장하지 마세요.
